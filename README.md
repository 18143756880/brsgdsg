# JavaScript 基础知识练习

这是一个用于练习 JavaScript 基础知识的 HTML 页面，涵盖了以下核心概念：

## 📚 内容涵盖

### 1. 变量的定义
- `var` - 函数作用域变量
- `let` - 块级作用域变量
- `const` - 常量声明

### 2. 数据类型
- 基本类型：String、Number、Boolean、Undefined、Null、Symbol、BigInt
- 引用类型：Object、Array、Function

### 3. 表达式与运算符
- 算术运算符：`+`、`-`、`*`、`/`、`%`、`**`
- 比较运算符：`>`、`<`、`===`、`==` 等
- 逻辑运算符：`&&`、`||`、`!`
- 赋值运算符：`=`、`+=`、`-=`、`*=` 等

### 4. 流程控制
- 条件语句：`if-else`、`switch`
- 循环语句：`for`、`while`、`for...of`

### 5. 函数
- 函数声明
- 函数表达式
- 箭头函数
- 默认参数
- 剩余参数
- 回调函数

## 🚀 使用方法

1. 直接在浏览器中打开 `javascript-practice.html` 文件
2. 点击各个部分的"运行示例"按钮查看代码执行结果
3. 查看页面上的代码示例和输出结果

- `javascript-practice.html` - 主要的练习页面，包含完整的 HTML、CSS 和 JavaScript 代码

## 📝 练习内容

页面包含六个主要部分：
1. 变量的定义 - 演示三种变量声明方式的区别
2. 数据类型 - 展示 JavaScript 的 8 种数据类型
3. 表达式与运算符 - 各种运算符的使用示例
4. 流程控制 - 条件判断和循环语句
5. 函数 - 多种函数定义方式的对比
6. 综合练习 - 学生成绩管理系统示例

## 🎯 学习目标

通过本页面的练习，你将掌握：
- 如何正确声明和使用变量
- 理解 JavaScript 的数据类型系统
- 熟练使用各种运算符
- 掌握流程控制语句的使用
- 学会定义和调用函数
- 能够综合运用所学知识解决实际问题
